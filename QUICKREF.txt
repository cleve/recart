🔧 GETTING STARTED WITH DEVELOPMENT
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Add a new weapon system:
    → Open spaceship_simulator/ship/ship.py
    → Add weapon instance in ShipState.__init__()
    → Wire up input in input_handler.py
    → Update HUD in hud/panel.py

  Add a new ship system:
    → Add property in ship.py
    → Create _update_system() method
    → Call in ship.update()
    → Display in HUD

  See DEVELOPMENT.md for detailed examples
